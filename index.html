Portifólio

PROARKY [ HTML5 | CSS3 | JS | PHP | MYSQL ]


Proarky é um sistema web responsivo e especializado que resolve um problema crítico para profissionais de engenharia e arquitetura: a complexa gestão de processos de legalização de imóveis e solicitação de alvarás junto aos órgãos municipais. O sistema, atualmente como MVP, foi concebido para organizar de forma simplificada todos os processos sob responsabilidade do profissional, e inclui a capacidade de realizar o cálculo imediato das taxas existentes, permitindo que o valor seja repassado aos clientes de forma ágil e transparente, otimizando o fluxo de trabalho e a satisfação do cliente.

SCRIPTORIUM [ HTML5 | CSS3 | JS | PHP | MYSQL ]


Scriptorium é uma plataforma web concebida para superar um desafio real da gestão de um extenso acervo pessoal de livros físicos, simplificando drasticamente o controle de empréstimos e devoluções para colecionadores individuais. O sistema oferece: 1. Acesso com permissões segregadas para administradores e leitores. 2. Modo leitura gamificado, que incentiva o engajamento. 3. Calculo de tempo estimado para a leitura de cada título, ajudando os leitores a alcançarem suas metas de forma eficaz.
